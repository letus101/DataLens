Théorie

La loi de Bernoulli modélise une expérience aléatoire n'ayant que deux issues possibles : succès (1) ou échec (0).

Pour une variable aléatoire X suivant une loi de Bernoulli de paramètre p :

\[ P(X = 1) = p \]

\[ P(X = 0) = 1 - p \]

Où p est la probabilité de succès, avec 0 ≤ p ≤ 1

Caractéristiques :

  • Espérance : E(X) = p
  • Variance : V(X) = p(1-p)
  • Écart-type : σ = \(\sqrt{p(1-p)}\)

Applications :

  • Pile ou face
  • Succès/échec d'un test
  • Réussite/échec d'un événement
