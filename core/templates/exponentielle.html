Théorie

La loi exponentielle est utilisée pour modéliser le temps d'attente jusqu'à la prochaine occurrence d'un événement dans un processus de Poisson.

Pour une variable aléatoire X suivant une loi exponentielle de paramètre λ :

Fonction de densité :

\[ f(x) = \lambda e^{-\lambda x} \quad \text{pour } x \geq 0 \]

Fonction de répartition :

\[ F(x) = 1 - e^{-\lambda x} \quad \text{pour } x \geq 0 \]

Où λ > 0 est le paramètre d'intensité

Caractéristiques :

  • Espérance : E(X) = 1/λ
  • Variance : V(X) = 1/λ²
  • Écart-type : σ = 1/λ
  • Médiane : ln(2)/λ

Propriété d'absence de mémoire :

La loi exponentielle est la seule loi continue possédant la propriété de "perte de mémoire" :

\[ P(X > s + t | X > s) = P(X > t) \]

Applications :

  • Durée de vie des composants électroniques
  • Temps d'attente entre deux événements rares
  • Analyse de fiabilité
  • Files d'attente
