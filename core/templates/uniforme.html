Uniform Distribution

Théorie

La loi uniforme est la distribution la plus simple où chaque valeur dans un intervalle [a,b] a la même probabilité d'apparition.

Pour une variable aléatoire X suivant une loi uniforme sur [a,b] :

Fonction de densité :

\[ f(x) = \begin{cases} \frac{1}{b-a} & \text{pour } x \in [a,b] \\ 0 & \text{sinon} \end{cases} \]

Fonction de répartition :

\[ F(x) = \begin{cases} 0 & \text{pour } x < a \\ \frac{x-a}{b-a} & \text{pour } x \in [a,b] \\ 1 & \text{pour } x > b \end{cases} \]

Caractéristiques :

  • Espérance : E(X) = \(\frac{a+b}{2}\)
  • Variance : V(X) = \(\frac{(b-a)^2}{12}\)
  • Écart-type : σ = \(\frac{b-a}{\sqrt{12}}\)
  • Médiane = \(\frac{a+b}{2}\)

Propriétés :

  • Distribution symétrique
  • Densité constante sur l'intervalle
  • Maximum d'entropie pour un intervalle borné

Applications :

  • Génération de nombres aléatoires
  • Erreurs d'arrondi
  • Temps d'attente aléatoire
  • Position aléatoire dans un intervalle
  • Simulations Monte Carlo
