Théorie

La loi de Poisson modélise le nombre d'occurrences d'un événement rare dans un intervalle de temps ou d'espace donné.

Pour une variable aléatoire X suivant une loi de Poisson de paramètre λ :

\[ P(X = k) = \frac{\lambda^k e^{-\lambda}}{k!} \quad \text{pour } k \in \mathbb{N} \]

Où :

  • λ est le paramètre d'intensité (nombre moyen d'occurrences)
  • k est le nombre d'occurrences
  • e est la constante d'Euler (≈ 2.71828)

Caractéristiques :

  • Espérance : E(X) = λ
  • Variance : V(X) = λ
  • Écart-type : σ = \(\sqrt{\lambda}\)

Propriétés :

  • Somme de variables de Poisson indépendantes suit une loi de Poisson
  • Approximation de la loi binomiale quand n est grand et p petit
  • Lien avec la loi exponentielle pour les temps d'attente

Applications :

  • Nombre d'appels téléphoniques par heure
  • Nombre de défauts dans une production
  • Nombre d'accidents par jour
  • Mutations génétiques dans un segment d'ADN
  • Files d'attente et théorie des files
